Data Types:
* All (all) - 
* Player Positions (position) - 
* NFL Teams (teams) - 
* Players (players) -
* Games (games) - 


```
$ nfl_data collect --type=<Data Type> --start=<Start Year> --end=<End Year>
```

```
nfl_data export --type=<Data Type> --dir=<Desitnation directory> [--force]
```
